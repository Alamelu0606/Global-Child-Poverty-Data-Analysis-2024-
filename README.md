# Global-Child-Poverty-Data-Analysis-2024-
A comprehensive data analytics project analyzing Global and Regional Trends in Moderate Child Food Poverty (2005–2024) using UNICEF’s 2024 dataset. The project integrates Excel for data cleaning and transformation with Power BI for advanced visualization, DAX modelling, and dashboard storytelling
Repository Overview

This repository contains all project files, including:

Excel Data Preparation – Unpivoted and cleaned UNICEF dataset.
Power BI Dashboard – Interactive visuals, slicers, and KPIs.
DAX Measures – Custom calculations for reliability, severity, and poverty insights.
Final Report (Word/PDF) – Comprehensive documentation with insights and conclusions.

Key Features:

Regional Insights Dashboard: Shows child food poverty trends across continents (Africa, Asia, Europe, etc.).
Demographic Analysis: Compares gender, urban–rural residence, education level, and income quintiles.
Interactive Navigation: Drill-through visuals with slicers for confidence levels and poverty categories.
Predictive & Prescriptive Insights: Forecasts trends and recommends targeted interventions.
Clean UI:  Theme with soft pastel visuals and accessible design.

Outcomes

Identified high-poverty regions such as Africa and South Asia.
Highlighted maternal education and income levels as major determinants of child nutrition.
Created a data-driven visualization platform to support policy-making and social program planning.
